# Code of Conduct

This project adheres to a code of conduct to ensure a welcoming environment for all contributors.

## Our Standards

- Be respectful and inclusive
- Use welcoming and inclusive language
- Be collaborative
- Show empathy towards other community members

## Unacceptable Behavior

- Harassment or discrimination
- Offensive comments
- Personal attacks

## Enforcement

Violations of this code of conduct may result in removal of contributions or discussion from the community.

## Contact

If you witness or experience a violation of this code of conduct, please contact the project maintainers.

Thank you for helping maintain a welcoming and respectful community.
