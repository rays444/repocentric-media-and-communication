# Media and Communication Report

This repository contains the academic report on Media and Communication, structured into units covering various aspects of the field.

## Overview

The report explores the foundations, types, systems, ethics, journalism, digital media, and public relations.

## Structure

- [00_Introduction](00_Introduction/): Main report introduction
- [01_Principles_of_Mass_Communication](01_Principles_of_Mass_Communication/): Foundations of Communication
- [02_Media_and_Society](02_Media_and_Society/): Types of Communication
- [03_Introduction_to_Communication_Studies](03_Introduction_to_Communication_Studies/): Media Systems and Institutions
- [04_History_of_Mass_Communication](04_History_of_Mass_Communication/): Media Ethics and Law
- [05_English_for_Mass_Communication](05_English_for_Mass_Communication/): Journalism and Reporting
- [06_Critical_Thinking_and_Problem_Solving](06_Critical_Thinking_and_Problem_Solving/): Digital and Social Media
- [07_Communication_and_Collaboration_Skills](07_Communication_and_Collaboration_Skills/): Public Relations and Advertising
- [Appendices](Appendices/): Additional resources including glossary, key theories, and extra resources
